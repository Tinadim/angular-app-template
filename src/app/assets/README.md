# Assets

This folder contains all static files served by the application, such as images, fonts and styles

## Project Structure

```
- fonts                 // fonts used by the Angular app
- i18n                  // translation files to be used by the project and allow internationalization
- img                   // images used by the Angular app
- styles                // styles sharead across the application such as font sizes, etc
    - buttons.scss      // styles applied to app buttons
    - fonts.scss        // styles applied to fonts (will like declare classes for icon fonts)
    - forms.scss        // styles for form elements, like input fields
    - layers.scss       // styles for application layers, to control the Z-indexes of components in the app 
    - modals.scss       // styles for application modals
    - panels.scss       // styles for application panels
    - styles.scss       // global styles such as ones that apply to the body or the html itself
    - text.scss         // styles for application text, like headers font sizes and margins
    - theme.scss        // themes to override default application colors and styles
- favicon.ico           // the app favicon
- index.html            // main entry point of the application
- README.md
```
